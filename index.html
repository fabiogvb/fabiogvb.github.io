Monitor e alarme do uso de memoria RAM em niveis criticos estabelicidos pelo usuario

Escrito em Shell script esse monitor emite alerta sonoro e visual em sistemas linux com interfaces graficas quando o uso da memoria RAM atinge niveis críticos.

Re-ordenador de contigs genomicos

Escrito em Shell script esse re-ordenador de contigs genomicos para arquivos no formato GenBank para gera um unico cromossomo artificial baseado em um genoma proximo completo de referencia fornecido pelo usuario.

Compressed CoPy

Esse pequeno script escrito em Shell script, serve para transferir grandes arquivos dentro de uma mesma rede, ele abre uma porta e realiza a transferencia de arquivos ou diretorios, compactando os dados em tempo de execucao, e dessa forma reduzindo muito o tempo de copia, e sem a necessidade de ocupar espaço em disco com arquivos compactados no computador de origem. O ccp utiliza sintaxe similar ao famoso scp.

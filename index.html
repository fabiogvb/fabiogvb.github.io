Alguns projetos simples online:


Preditor de proteínas de membrana

Preditor de proteínas de membrana escrito em python usando flask e o modelo de machine learning XGBOOST treinado com proteínas transmembranas e citoplasmáticas de bactérias obtidas da base de dados curada do uniprot.

Reordenador de contigs genômicos

Escrito em bash script esse reordenador de contigs genômicos para arquivos no formato GenBank para gera um único cromossomo artificial baseado em um genoma próximo completo de referência fornecido pelo usuário.

Compressed CoPy

Esse pequeno script escrito em bash, é ideal pra transferir grandes arquivos ou dados dentro de uma mesma rede, ele abre uma porta e realiza a transferência de arquivos ou diretórios, compactando os dados em tempo de execução, e desse forma reduzindo o tempo de cópia, e sem a necessidade de ocupar espaço em disco com arquivos compactados. O ccp utiliza sintaxe similar ao famoso scp.

NiceAdmin template

Bootstrap template e formulários interativos em JavaScript, CSS e HTML.
